This module serves as the key input of entire system. It is suppose to contain two major functions
that ideally enable users to download data via Chinese stock database Wind or Tinysoft. Of course, 
it requires access ID and passwords to get through the system, which should be provided by users
in the form of csv or txt documents. 

Besides account information, user should also define the time horizon, data key and stock tickers.
Data download by sector might be provided in later updates.

Also, a doorway of data saving is kept but leave unwritten, just to handle the future request of data
localization.
